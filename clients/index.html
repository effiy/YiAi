🤖 YiAi

AI 服务平台客户端

💬
AI 对话客户端
使用流式 API 与 AI 模型进行对话，支持多轮对话、图片上传和聊天记录管理。支持语义搜索和历史记录查看。
🗄️
MongoDB 管理
MongoDB 数据库管理工具，支持集合操作、文档查询、数据导入导出等功能。
📦
OSS 文件管理
对象存储服务管理工具，支持文件上传、下载、列表查看和删除等操作。
⚙️
模块动态调用
动态执行 Python 模块和方法，支持参数传递和结果查看。适用于爬虫、文件处理等模块调用。
🧠
Mem0 记忆管理
Mem0 记忆管理系统，支持添加、搜索、更新和删除记忆。使用向量数据库实现语义搜索功能。
🔍
Qdrant 向量数据库
Qdrant 向量数据库管理工具，支持集合管理、向量插入、搜索、更新和删除等操作。
