# 路由规范

本文档定义 YiAi 项目的路由设计规范。

---

## 路由组织

### 使用 APIRouter

每个功能模块使用独立的 APIRouter：

```python
# src/api/routes/users.py
from fastapi import APIRouter

router = APIRouter(prefix="/users", tags=["Users"])

@router.get("")
async def list_users():
    pass

@router.get("/{user_id}")
async def get_user(user_id: str):
    pass
```

### 路由文件命名

使用功能名称命名路由文件：

```
src/api/routes/
├── execution.py      # 执行模块路由
├── upload.py         # 上传路由
├── wework.py         # 企业微信路由
└── maintenance.py    # 维护路由
```

### 路由注册

在 `src/main.py` 中统一注册路由：

```python
from api.routes import upload, execution, wework, maintenance

app = FastAPI()

app.include_router(upload.router, tags=["Upload"])
app.include_router(execution.router, tags=["Execution"])
app.include_router(wework.router, tags=["WeWork"])
app.include_router(maintenance.router, tags=["Maintenance"])
```

## 路径设计

### 前缀使用

使用 `prefix` 参数设置路由前缀：

```python
router = APIRouter(prefix="/api/v1/users")
```

### 路径参数

使用花括号 `{}` 表示路径参数：

```python
@router.get("/{user_id}")
async def get_user(user_id: str):
    pass

@router.get("/{user_id}/orders/{order_id}")
async def get_user_order(user_id: str, order_id: str):
    pass
```

### 路径格式

- 使用小写字母
- 使用连字符 `-` 分隔单词
- 使用名词复数形式

```python
# 正确示例
router = APIRouter(prefix="/user-profiles")

# 避免示例
router = APIRouter(prefix="/UserProfiles")
router = APIRouter(prefix="/user_profiles")
```

## 标签和描述

### 路由标签

使用 `tags` 参数对路由进行分类：

```python
router = APIRouter(tags=["用户管理"])
```

### 端点描述

使用 `summary` 和 `description` 提供端点文档：

```python
@router.get(
    "/{user_id}",
    summary="获取用户详情",
    description="根据用户 ID 获取用户的详细信息"
)
async def get_user(user_id: str):
    pass
```

## 响应模型

### 使用响应模型

使用 `response_model` 指定响应类型：

```python
from models.schemas import UserResponse

@router.get("/{user_id}", response_model=UserResponse)
async def get_user(user_id: str) -> UserResponse:
    user = await user_service.get_by_id(user_id)
    return UserResponse.model_validate(user)
```

### 状态码指定

使用 `status_code` 指定成功响应的状态码：

```python
from fastapi import status

@router.post("", status_code=status.HTTP_201_CREATED)
async def create_user(request: UserCreateRequest):
    pass
```

## 依赖注入

### 路由依赖

使用 `dependencies` 参数添加路由级别的依赖：

```python
from fastapi import Depends
from core.middleware import verify_token

router = APIRouter(
    dependencies=[Depends(verify_token)]
)
```

### 端点依赖

在单个端点上使用依赖：

```python
@router.get("/{user_id}")
async def get_user(
    user_id: str,
    current_user: User = Depends(get_current_user)
):
    pass
```

## 错误处理

### HTTP 异常

使用 `HTTPException` 处理 HTTP 错误：

```python
from fastapi import HTTPException, status

@router.get("/{user_id}")
async def get_user(user_id: str):
    user = await user_service.get_by_id(user_id)
    if not user:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail="用户不存在"
        )
    return user
```

### 业务异常

使用项目的业务异常：

```python
from core.exceptions import BusinessException
from core.error_codes import ErrorCode

@router.post("")
async def create_user(request: UserCreateRequest):
    existing = await user_service.get_by_email(request.email)
    if existing:
        raise BusinessException(
            ErrorCode.DATA_ALREADY_EXISTS,
            message="邮箱已被注册"
        )
    pass
```

## 路由测试

### 测试路由

使用 TestClient 测试路由：

```python
from fastapi.testclient import TestClient
from src.main import app

client = TestClient(app)

def test_get_user():
    response = client.get("/users/test-id")
    assert response.status_code == 200
```
