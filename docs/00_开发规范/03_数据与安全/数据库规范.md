# 数据库规范

本文档定义 YiAi 项目的 MongoDB 数据库使用规范。

---

## 数据库配置

### 连接配置

MongoDB 连接通过 `config.yaml` 配置：

```yaml
mongodb:
  url: "mongodb://localhost:27017"
  db_name: "ruiyi"
  pool_size: 10
  max_pool_size: 50
```

### 集合名称配置

集合名称也通过配置管理：

```yaml
collection:
  sessions: "sessions"
  rss: "rss"
  chat_records: "chat_records"
  oss_file_info: "oss_file_info"
  oss_file_tags: "oss_file_tags"
  pet_data_sync: "pet_data_sync"
  seeds: "seeds"
```

## 数据库单例模式

### 获取数据库实例

项目使用单例模式管理数据库连接：

```python
# 正确示例
from core.database import db

async def get_user(user_id: str):
    await db.initialize()  # 确保已初始化
    return await db.find_one("sessions", {"id": user_id})
```

### 初始化时机

数据库在应用启动时初始化：

```python
# src/main.py
@asynccontextmanager
async def lifespan(app: FastAPI):
    # 启动时
    if settings.startup_init_database:
        await db.initialize()
        logger.info("数据库连接已初始化")

    yield

    # 关闭时
    if db.client:
        db.client.close()
        logger.info("数据库连接已关闭")
```

## 集合设计规范

### 集合命名

- 使用小写字母和下划线
- 使用复数形式
- 使用配置中的集合名称，不要硬编码

```python
# 正确示例
collection_name = settings.collection_sessions
await db.insert_one(collection_name, document)

# 避免示例
await db.insert_one("Sessions", document)  # 大写
await db.insert_one("session", document)   # 单数
```

### 文档结构

#### 必须字段

每个文档应该包含以下字段：

```python
document = {
    "_id": ObjectId,          # MongoDB 自动生成
    "id": str,                # 业务 ID（推荐）
    "created_at": datetime,    # 创建时间
    "updated_at": datetime,    # 更新时间（可选）
}
```

#### 时间字段

使用 UTC 时间：

```python
from datetime import datetime, timezone

document = {
    "id": "user-001",
    "created_at": datetime.now(timezone.utc),
    "updated_at": datetime.now(timezone.utc),
}
```

## 数据操作规范

### 使用统一的访问方法

使用 `core.database` 提供的统一方法：

```python
# 正确示例
from core.database import db

# 查询单条
user = await db.find_one(settings.collection_sessions, {"id": user_id})

# 查询多条
users = await db.find_many(
    settings.collection_sessions,
    {"status": "active"},
    limit=100
)

# 插入
user_id = await db.insert_one(settings.collection_sessions, user_data)

# 更新
result = await db.update_one(
    settings.collection_sessions,
    {"id": user_id},
    {"$set": {"name": "New Name"}}
)

# 删除
deleted = await db.delete_one(settings.collection_sessions, {"id": user_id})
```

### 查询规范

#### 索引利用

确保查询使用索引：

```python
# 正确示例 - 使用索引字段查询
await db.find_one(settings.collection_sessions, {"id": user_id})

# 避免示例 - 全表扫描
await db.find_one(settings.collection_sessions, {"metadata.field": value})
```

#### 限制结果数量

查询多条时指定 limit：

```python
# 正确示例
await db.find_many(
    settings.collection_chat_records,
    {"user_id": user_id},
    limit=100
)

# 避免示例 - 不限制数量
await db.find_many(
    settings.collection_chat_records,
    {"user_id": user_id}
)
```

#### 投影字段

只查询需要的字段：

```python
# 正确示例 - 使用投影
await db.find_one(
    settings.collection_sessions,
    {"id": user_id},
    projection={"name": 1, "email": 1, "_id": 0}
)
```

### 插入规范

```python
# 正确示例
from datetime import datetime, timezone

user_data = {
    "id": str(uuid.uuid4()),
    "name": "Test User",
    "email": "test@example.com",
    "created_at": datetime.now(timezone.utc),
}

user_id = await db.insert_one(settings.collection_sessions, user_data)
```

### 更新规范

```python
# 正确示例 - 使用 $set
await db.update_one(
    settings.collection_sessions,
    {"id": user_id},
    {
        "$set": {
            "name": "New Name",
            "updated_at": datetime.now(timezone.utc),
        }
    }
)

# 避免示例 - 替换整个文档
await db.update_one(
    settings.collection_sessions,
    {"id": user_id},
    new_document
)
```

### 删除规范

```python
# 正确示例 - 软删除（如适用）
await db.update_one(
    settings.collection_sessions,
    {"id": user_id},
    {"$set": {"deleted": True, "deleted_at": datetime.now(timezone.utc)}}
)

# 或硬删除
deleted_count = await db.delete_one(settings.collection_sessions, {"id": user_id})
```

## 事务规范

### 使用事务

需要原子性的操作使用事务：

```python
async def transfer_funds(from_id: str, to_id: str, amount: float):
    """转账操作 - 使用事务"""
    session = await db.client.start_session()
    try:
        async with session.start_transaction():
            # 扣款
            await db.update_one(
                "accounts",
                {"id": from_id, "balance": {"$gte": amount}},
                {"$inc": {"balance": -amount}},
                session=session
            )
            # 加款
            await db.update_one(
                "accounts",
                {"id": to_id},
                {"$inc": {"balance": amount}},
                session=session
            )
            logger.info(f"转账成功: {from_id} -> {to_id}, amount={amount}")
    except Exception as e:
        await session.abort_transaction()
        logger.error(f"转账失败: {e}", exc_info=True)
        raise
    finally:
        await session.end_session()
```

## 性能优化规范

### 批量操作

使用批量操作减少往返：

```python
# 正确示例 - 批量插入
await db.insert_many(settings.collection_sessions, user_list)

# 避免示例 - 循环插入
for user in user_list:
    await db.insert_one(settings.collection_sessions, user)
```

### 索引创建

为常用查询字段创建索引：

```python
# 在应用启动时创建索引
async def create_indexes():
    """创建数据库索引"""
    collection = db.db[settings.collection_sessions]
    await collection.create_index("id", unique=True)
    await collection.create_index("email")
    await collection.create_index("created_at")

    collection = db.db[settings.collection_chat_records]
    await collection.create_index("user_id")
    await collection.create_index("created_at")
```
