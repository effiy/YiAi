# 核心功能

本目录包含 YiAi 项目的核心功能详细说明。

---

## 📋 功能列表

### 11_AI聊天
- [多轮对话](./11_AI聊天/多轮对话.md) - 集成 Ollama 本地 AI 模型，支持上下文感知的多轮对话
- [历史记录](./11_AI聊天/历史记录.md) - 自动保存聊天会话和消息记录，支持随时回溯

### 12_RSS管理
- [定时抓取](./12_RSS管理/定时抓取.md) - 基于 APScheduler 的自动定时 RSS 源内容抓取
- [多源管理](./12_RSS管理/多源管理.md) - 支持配置多个 RSS 源，统一存储和管理文章内容

### 13_文件存储
- [双重存储](./13_文件存储/双重存储.md) - 支持阿里云 OSS 和本地静态文件存储两种方案，自动 fallback
- [文件管理](./13_文件存储/文件管理.md) - 提供文件上传、读取、写入、删除、重命名等完整操作
- [图片清理](./13_文件存储/图片清理.md) - 自动扫描并清理未被引用的图片文件，释放存储空间

### 14_动态执行
- [模块执行引擎](./14_动态执行/模块执行引擎.md) - 通过 REST API 动态执行 Python 模块方法，支持同步/异步函数
- [SSE流式传输](./14_动态执行/SSE流式传输.md) - 为生成器函数提供 Server-Sent Events 流式响应
- [白名单控制](./14_动态执行/白名单控制.md) - 通过配置白名单限制可执行的模块，确保安全性

### 15_安全认证
- [令牌认证](./15_安全认证/令牌认证.md) - 可选的 X-Token 认证中间件，保护 API 端点
- [异常处理](./15_安全认证/异常处理.md) - 统一的异常处理机制，标准化错误响应格式

### 16_企业集成
- [企业微信](./16_企业集成/企业微信.md) - 支持通过 Webhook 发送消息到企业微信群聊
