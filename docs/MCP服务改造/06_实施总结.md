## §2 AI 调用时序图

```mermaid
sequenceDiagram
    participant Client as AI 客户端 (Cursor/Claude)
    participant MCP as FastApiMCP (/mcp)
    participant FastAPI as FastAPI 应用
    participant Route as API 路由

    Client->>MCP: SSE 连接请求 GET /mcp
    MCP-->>Client: event: endpoint<br/>data: /mcp/messages/?session_id=xxx

    Client->>MCP: JSON-RPC initialize
    MCP-->>Client: 返回工具列表 (12 个工具)

    Client->>MCP: tools/call (如 read_file)
    MCP->>FastAPI: ASGI 内部调用 POST /read-file
    FastAPI->>Route: 路由分发
    Route->>Route: Pydantic 模型验证
    Route->>Route: 执行业务逻辑
    Route-->>FastAPI: 返回 StandardResponse
    FastAPI-->>MCP: 返回响应
    MCP-->>Client: 返回工具调用结果 (JSON)
```

**说明**：工具调用通过 ASGI 内部通信直接转发到 FastAPI 路由，无额外 HTTP 开销。Maintenance 标签端点已被过滤排除。
