## 变更对比

### src/main.py

**变更类型**：修改

**变更前**：
```python
    app = FastAPI(
        title="YiAi API",
        description="YiPet AI 服务 API",
        version="1.0.0",
        lifespan=_build_lifespan(db_init_enabled, rss_init_enabled)
    )
    # ... 路由注册、CORS、认证、静态文件 ...
    return app
```

**变更后**：
```python
    from fastapi_mcp import FastApiMCP

    app = FastAPI(
        title="YiAi API",
        description="YiPet AI 服务 API",
        version="1.0.0",
        lifespan=_build_lifespan(db_init_enabled, rss_init_enabled)
    )
    # ... 路由注册、CORS、认证、静态文件 ...

    mcp = FastApiMCP(
        app,
        name="YiAi MCP",
        describe_all_responses=True,
        describe_full_response_schema=True,
        exclude_tags=["Maintenance"]
    )
    mcp.mount()

    return app
```

**一句话说明**：在 FastAPI 应用创建和路由注册后，添加 FastApiMCP 实例创建和挂载，使 `/mcp` 端点可用。

### requirements.txt

**变更类型**：新增行

**变更前**：
```
fastapi>=0.104.0
uvicorn>=0.24.0
...
```

**变更后**：
```
fastapi>=0.104.0
uvicorn>=0.24.0
...
fastapi-mcp>=0.1.0
```

**一句话说明**：新增 fastapi-mcp 依赖，用于 MCP 服务器功能。

### src/api/routes/execution.py

**变更类型**：修改（路由装饰器参数）

**变更前**：
```python
@router.get("/")
async def execute_module_via_get(...):
    ...

@router.post("/")
async def execute_module_via_post(...):
    ...
```

**变更后**：
```python
@router.get("/", operation_id="execute_module_get")
async def execute_module_via_get(...):
    ...

@router.post("/", operation_id="execute_module_post")
async def execute_module_via_post(...):
    ...
```

**一句话说明**：为路由添加显式 operation_id，使 MCP 工具名称清晰直观。
