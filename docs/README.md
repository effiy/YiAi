# YiAi 项目文档

欢迎使用 YiAi 项目文档！

---

## 📋 项目概述

YiAi 是一个基于 FastAPI 的 AI 服务后端，通过 REST API 端点提供丰富的功能。它采用模块化设计，支持动态扩展，集成了 AI 聊天、RSS 管理、文件存储等多种能力。

---

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 配置

1. 复制并编辑 `config.yaml` 配置文件
2. 根据需要配置各项参数
3. 环境变量可以覆盖 YAML 配置（大写、蛇形命名，如 `SERVER_HOST`）

详细的配置说明请参考 [配置指南](./配置指南.md)。

### 启动开发服务器

```bash
python main.py
```

服务器将在 http://localhost:8000 启动，已启用自动重载。

---

## ✨ 核心功能

- 🤖 **AI 聊天服务** - 集成 Ollama 本地 AI 模型，支持多轮对话和历史记录
- 📰 **RSS 源管理** - 自动定时抓取 RSS 源内容，支持多源管理
- 📤 **文件上传与存储** - 支持 OSS 和本地存储双重方案，完整的文件管理功能
- ⚡ **动态模块执行引擎** - 通过 REST API 动态执行 Python 模块方法
- 🛡️ **安全与认证** - 可选的令牌认证，统一的异常处理

详细的功能介绍请参考 [核心功能](./核心功能/README.md)。

## 🌐 API 端点

YiAi 提供丰富的 REST API 端点，涵盖模块执行、文件管理、企业微信集成和系统维护等功能。

- ⚡ **执行模块** - 动态执行 Python 模块方法的核心扩展机制，支持同步/异步函数和生成器流式响应
- 📤 **文件上传与管理** - 完整的文件管理功能，支持 OSS 和本地存储双重方案，提供上传、读取、写入、删除等操作
- 🏢 **企业微信** - 企业微信机器人 Webhook 集成，支持向企业微信群发送消息
- 🔧 **维护工具** - 系统维护和清理工具，支持清理未引用的图片等功能

完整的 API 端点文档请参考 [API端点](./API端点.md)。

## 📊 数据库集合

YiAi 使用 MongoDB 存储各类业务数据，包括用户会话、RSS 文章、聊天记录、文件信息等。

**主要集合：**
- **sessions** - 用户会话数据
- **rss** - RSS 源文章内容
- **chat_records** - AI 聊天历史记录
- **oss_file_info** - 文件元数据信息
- **oss_file_tags** - 文件标签分类

详细的数据库集合说明请参考 [数据库集合](./数据库集合.md)。
