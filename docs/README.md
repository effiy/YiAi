## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动开发服务器

```bash
python main.py
```

服务器将在 http://localhost:8000 启动，API 文档可通过 `/docs` 和 `/redoc` 访问。
