# 安全规范

本文档定义 YiAi 项目的安全开发规范。

---

## 认证与授权

### 认证中间件

项目支持可选的令牌认证：

```python
# src/core/middleware.py
@app.middleware("http")
async def header_verification_middleware(request: Request, call_next):
    if not settings.middleware_auth_enabled:
        return await call_next(request)

    token = request.headers.get("X-Token")
    expected_token = settings.auth_token

    if not token or token != expected_token:
        return JSONResponse(
            status_code=status.HTTP_401_UNAUTHORIZED,
            content={"code": 401, "message": "未授权"}
        )

    return await call_next(request)
```

### 配置认证

在 `config.yaml` 中配置：

```yaml
middleware:
  auth_enabled: true
  auth_token: "your-secret-token-change-me"
```

认证令牌也可以通过环境变量 `API_X_TOKEN` 设置。

## 输入验证

### 使用 Pydantic 验证

所有输入必须使用 Pydantic 模型验证：

```python
from pydantic import BaseModel, Field, EmailStr, validator

class UserCreateRequest(BaseModel):
    name: str = Field(..., min_length=1, max_length=100, description="用户名")
    email: EmailStr = Field(..., description="邮箱地址")
    age: int = Field(..., ge=0, le=150, description="年龄")

    @validator("name")
    def name_not_contain_special_chars(cls, v):
        if any(c in v for c in ["<", ">", "'", '"', ";"]):
            raise ValueError("用户名包含非法字符")
        return v.strip()
```

### 路径验证

文件路径必须验证，防止路径遍历攻击：

```python
def _validate_path(path: str, param_name: str = "路径") -> str:
    """验证路径安全性，返回规范化的相对路径"""
    if not path or ".." in path or path.startswith("/"):
        raise BusinessException(ErrorCode.INVALID_PARAMS, message=f"非法{param_name}")
    return path.strip().replace("\\", "/")

def _resolve_static_path(target_file: str) -> str:
    """将相对路径解析为安全的绝对路径"""
    rel = (target_file or "").strip().replace("\\", "/")
    if rel.startswith("static/"):
        rel = rel[7:]
    if not rel or rel.startswith("/") or ".." in rel:
        raise BusinessException(ErrorCode.INVALID_PARAMS, message="非法路径")

    base_dir = os.path.realpath(os.path.abspath(settings.static_base_dir))
    abs_path = os.path.realpath(os.path.abspath(os.path.join(base_dir, os.path.normpath(rel))))

    if os.path.commonpath([base_dir, abs_path]) != base_dir:
        raise BusinessException(ErrorCode.INVALID_PARAMS, message="非法路径")

    return abs_path
```

## 输出编码

### XSS 防护

输出用户输入时进行转义：

```python
import html

def escape_html(content: str) -> str:
    """转义 HTML 内容"""
    return html.escape(content)

# 使用示例
safe_content = escape_html(user_input)
return {"content": safe_content}
```

### 响应头安全

设置安全的响应头：

```python
# FastAPI 中间件中添加
from fastapi import Response

@app.middleware("http")
async def security_headers_middleware(request: Request, call_next):
    response = await call_next(request)
    response.headers["X-Content-Type-Options"] = "nosniff"
    response.headers["X-Frame-Options"] = "DENY"
    response.headers["X-XSS-Protection"] = "1; mode=block"
    return response
```

## 文件上传安全

### 文件类型验证

验证文件扩展名：

```python
ALLOWED_EXTENSIONS = {'.png', '.jpg', '.jpeg', '.gif', '.webp', '.svg', '.bmp', '.ico'}

def _is_image_file(filename: str) -> bool:
    """判断是否是图片文件"""
    if not filename:
        return False
    n = str(filename).lower()
    for ext in ALLOWED_EXTENSIONS:
        if n.endswith(ext):
            return True
    return False
```

### 文件大小限制

检查文件大小：

```python
MAX_FILE_SIZE = 50 * 1024 * 1024  # 50MB

if len(content) > MAX_FILE_SIZE:
    raise BusinessException(ErrorCode.INVALID_PARAMS, message="文件超过大小限制")
```

### 文件名清理

清理文件名，防止路径遍历：

```python
import re

def _normalize_no_spaces(value: str) -> str:
    return re.sub(r"\s+", "_", (value or "").strip())

def safe_filename(filename: str) -> str:
    """生成安全的文件名"""
    filename = _normalize_no_spaces(filename)
    # 移除危险字符
    filename = re.sub(r'[<>"|?*]', '', filename)
    # 确保有扩展名
    if not os.path.splitext(filename)[1]:
        filename += ".png"
    return filename
```

## NoSQL 注入防护

### 使用参数化查询

使用 Motor 的查询构建，不要拼接字符串：

```python
# 正确示例
await db.find_one(settings.collection_sessions, {"id": user_id})

# 避免示例 - 拼接查询（风险）
query = '{"id": "' + user_id + '"}'  # 不要这样做！
```

### 输入验证

验证查询输入：

```python
def validate_query_value(value: str) -> str:
    """验证查询值"""
    if any(c in value for c in ["$", "{", "}", "[", "]"]):
        raise BusinessException(ErrorCode.INVALID_PARAMS, message="非法查询参数")
    return value
```

## 敏感数据保护

### 配置中的敏感信息

敏感配置通过环境变量设置：

```yaml
# config.yaml 中可以留空
middleware:
  auth_token: ""

oss:
  access_key: ""
  secret_key: ""
```

通过环境变量覆盖：

```bash
export MIDDLEWARE_AUTH_TOKEN="your-secret-token"
export OSS_ACCESS_KEY="your-access-key"
export OSS_SECRET_KEY="your-secret-key"
```

### 日志中的敏感信息

不要在日志中记录敏感信息：

```python
# 正确示例
logger.info(f"用户登录成功: user_id={user_id}")

# 避免示例
logger.info(f"用户登录: password={password}")
logger.info(f"API 调用: token={token}")
```

### 密码存储

如果需要存储密码，使用安全的哈希：

```python
import hashlib
import os

def hash_password(password: str, salt: bytes = None) -> tuple[bytes, bytes]:
    """安全哈希密码"""
    if salt is None:
        salt = os.urandom(16)
    key = hashlib.pbkdf2_hmac(
        "sha256",
        password.encode("utf-8"),
        salt,
        100000
    )
    return salt, key

def verify_password(password: str, salt: bytes, key: bytes) -> bool:
    """验证密码"""
    _, test_key = hash_password(password, salt)
    return test_key == key
```

## CORS 配置

### 配置 CORS

在 `config.yaml` 中配置：

```yaml
cors:
  origins:
    - "https://your-domain.com"
    - "https://app.your-domain.com"
  allow_any_origin: false
```

### 避免通配符

生产环境不要使用通配符：

```yaml
# 生产环境 - 明确指定
cors:
  origins:
    - "https://your-domain.com"
  allow_any_origin: false

# 开发环境 - 可以使用通配符
cors:
  origins:
    - "*"
  allow_any_origin: true
```

## 错误信息安全

### 不暴露内部错误

返回给用户的错误信息不要包含内部细节：

```python
# 正确示例
try:
    result = await db.query(...)
except Exception as e:
    logger.error(f"数据库错误: {e}", exc_info=True)
    raise BusinessException(ErrorCode.INTERNAL_ERROR, message="服务器内部错误")

# 避免示例
try:
    result = await db.query(...)
except Exception as e:
    raise BusinessException(ErrorCode.INTERNAL_ERROR, message=f"数据库错误: {e}")
```
