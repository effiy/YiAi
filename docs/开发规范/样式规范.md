# 样式规范

本文档定义 YiAi 项目的代码样式和格式化规范。

---

## 代码格式化工具

### Black

使用 Black 进行代码格式化：

```bash
# 格式化代码
black src/

# 检查但不修改
black --check src/

# 显示差异
black --diff src/
```

Black 配置（pyproject.toml）：

```toml
[tool.black]
line-length = 100
target-version = ['py311']
include = '\.pyi?$'
```

### isort

使用 isort 进行导入排序：

```bash
# 排序导入
isort src/

# 检查但不修改
isort --check src/
```

isort 配置（pyproject.toml）：

```toml
[tool.isort]
profile = "black"
line_length = 100
multi_line_output = 3
include_trailing_comma = true
force_grid_wrap = 0
use_parentheses = true
ensure_newline_before_comments = true
```

## 导入排序规范

### 导入分组

导入按以下顺序分组：

1. 标准库导入
2. 第三方库导入
3. 本地包导入

每组之间空一行：

```python
# 正确示例
import os
import sys
from datetime import datetime
from typing import List, Optional

import aiohttp
from fastapi import APIRouter, Depends
from pydantic import BaseModel

from core.config import settings
from core.database import db
from models.schemas import UserResponse
```

### 导入格式

使用 `from ... import ...` 方式：

```python
# 正确示例
from core.config import settings
from models.schemas import User, UserCreate

# 避免示例
import core.config
import models.schemas
```

### 相对导入

在包内部使用相对导入：

```python
# 正确示例
from . import utils
from ..core import config

# 避免示例
from src.api import utils
from src.core import config
```

## 代码样式规范

### 行长度

行长度不超过 100 字符：

```python
# 正确示例
async def process_user_data(
    user_id: str,
    data: dict,
    options: Optional[dict] = None
) -> dict:
    """处理用户数据"""
    pass

# 避免示例
async def process_user_data(user_id: str, data: dict, options: Optional[dict] = None) -> dict:
    """处理用户数据"""
    pass
```

### 空行使用

- 函数之间空 2 行
- 类之间空 2 行
- 方法之间空 1 行
- 逻辑块之间空 1 行

```python
# 正确示例
class UserService:

    async def get_user(self, user_id: str):
        pass

    async def create_user(self, data: dict):
        pass


class OrderService:

    async def create_order(self, data: dict):
        pass
```

### 空格使用

运算符两侧加空格：

```python
# 正确示例
x = 1 + 2
name = "test"
if a and b:
    pass

# 避免示例
x=1+2
name="test"
if a and b:
    pass
```

逗号后加空格：

```python
# 正确示例
def func(a, b, c):
    pass

items = [1, 2, 3]

# 避免示例
def func(a,b,c):
    pass

items = [1,2,3]
```

## 字符串规范

### 引号使用

优先使用双引号，字符串内部包含双引号时使用单引号：

```python
# 正确示例
name = "Alice"
message = 'He said "Hello!"'

# 避免示例
name = 'Alice'
message = "He said \"Hello!\""
```

### 多行字符串

使用三引号：

```python
# 正确示例
long_text = """
这是一段
多行文本
"""

# 正确示例 - 保持缩进
long_text = (
    "这是一段"
    "多行文本"
)
```

### 字符串拼接

优先使用 f-string：

```python
# 正确示例
name = "Alice"
greeting = f"Hello, {name}!"

# 正确示例
greeting = "Hello, {}!".format(name)

# 避免示例
greeting = "Hello, " + name + "!"
```

## 注释规范

### 行内注释

行内注释使用 `#`，前面空 2 格：

```python
# 正确示例
total = price * quantity  # 计算总价

# 避免示例
total = price * quantity#计算总价
total = price * quantity  #计算总价
```

### 块注释

块注释缩进与代码一致：

```python
# 正确示例
# 验证用户输入
# 1. 检查必填字段
# 2. 验证数据格式
# 3. 检查业务规则
if not validate(data):
    raise ValueError()
```

## 文档字符串规范

### 函数文档字符串

使用 Google 风格：

```python
def calculate_total(price: float, quantity: int) -> float:
    """
    计算总价格。

    Args:
        price: 单价，必须为正数
        quantity: 数量，必须为正数

    Returns:
        总价格，等于 price * quantity

    Raises:
        ValueError: 当 price 或 quantity 为负数时

    Examples:
        >>> calculate_total(10.0, 2)
        20.0
    """
    if price < 0 or quantity < 0:
        raise ValueError("价格和数量必须为正数")
    return price * quantity
```

### 类文档字符串

```python
class UserService:
    """
    用户服务类。

    提供用户的增删改查等功能。

    Attributes:
        db: 数据库连接实例
        cache: 缓存实例
    """

    def __init__(self, db, cache):
        self.db = db
        self.cache = cache
```
