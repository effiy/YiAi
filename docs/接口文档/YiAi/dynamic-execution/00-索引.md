## 身份卡片

| 属性 | 值 |
|------|-----|
| 名称 | Dynamic Execution API |
| 路由前缀 | `/`（根路径，无前缀） |
| 标签 | `Execution` |
| 端点数 | 2（GET + POST） |
| 协议 | HTTP/1.1 + SSE（流式响应） |
| 认证 | X-Token（中间件层，白名单路径 `POST /upload-image-to-oss` 等可跳过） |

## 一句话定位

通用模块执行引擎——通过 HTTP 动态调用白名单内的任意 Python 模块函数，支持同步/异步/生成器/异步生成器四种返回模式，集成沙箱隔离和重入守卫。
