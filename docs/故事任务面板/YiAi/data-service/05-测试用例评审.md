## §2 测试用例

### 2.1 正常用例

| ID | Given | When | Then | 关联 FP | 优先级 |
|----|-------|------|------|---------|--------|
| TC-N1 | sessions 集合有 ≥3 条文档 | `query_documents({cname:"sessions", pageSize:2})` | 返回 `list` 长度 ≤2，`total` ≥3，不含 pageContent | FP1,FP3,FP5 | P0 |
| TC-N2 | sessions 集合有文档 | `query_documents({cname:"sessions", orderBy:"createdTime", orderType:"desc"})` | 按 createdTime 降序排列 | FP4 | P1 |
| TC-N3 | 传入合法 title 字段 | `query_documents({cname:"sessions", title:"测试"})` | 返回 title 含"测试"的文档（不区分大小写） | FP2 | P0 |
| TC-N4 | 传入合法 data dict | `create_document({cname:"sessions", data:{title:"test"}})` | 返回 `{key: <uuid>}`，可查询到文档且含 key/createdTime/updatedTime/order | FP6,FP1 | P0 |
| TC-N5 | 已知文档 key | `update_document({cname:"sessions", data:{key:<k>, title:"updated"}})` | 返回 `{key, updated:true}`，再查询 title 已变更 | FP7 | P0 |
| TC-N6 | filter 匹配已有文档 | `upsert_document({cname:"sessions", filter:{key:<k>}, update:{$set:{a:1}}})` | `matched_count=1, modified_count=1` | FP8 | P1 |
| TC-N7 | filter 不匹配任何文档 | `upsert_document({cname:"sessions", filter:{key:"nope"}, update:{$set:{a:1}}})` | `upserted_id` 非空，文档被创建 | FP8 | P1 |
| TC-N8 | 已知文档 key | `delete_document({cname:"sessions", key:<k>})` | 返回 `{key, deleted:true}`，再查询不存在 | FP9 | P0 |
| TC-N9 | 已知文档 key | `get_document_detail({cname:"sessions", id:<k>})` | 返回完整文档，不含 _id 和 pageContent | FP10 | P1 |
| TC-N10 | sessions 有 projectName+storyName 文档 | `list_story_task_dirs({})` | 返回去重目录列表，含 session_count 和 latest_time | FP11 | P1 |

### 2.2 边界用例

| ID | Given | When | Then | 关联 FP | 优先级 |
|----|-------|------|------|---------|--------|
| TC-B1 | cname 为空字符串 | `query_documents({cname:""})` | ValueError: 必须提供集合名称 | FP1 | P0 |
| TC-B2 | pageSize=0 | `query_documents({cname:"sessions", pageSize:0})` | pageSize 被 clamp 为 1 | FP3 | P1 |
| TC-B3 | pageSize=10000（超过上限） | `query_documents({cname:"sessions", pageSize:10000})` | pageSize 被 clamp 为 8000 | FP3 | P1 |
| TC-B4 | pageNum=0（小于 1） | `query_documents({cname:"sessions", pageNum:0})` | pageNum 被 clamp 为 1 | FP3 | P1 |
| TC-B5 | fields="title,name"（合法字段选择） | `query_documents({cname:"sessions", fields:"title,name"})` | 返回仅含 title/name/key 的文档 | FP5 | P1 |
| TC-B6 | excludeFields="extra"（合法排除） | `query_documents({cname:"sessions", excludeFields:"extra"})` | 返回不含 extra 字段的文档 | FP5 | P1 |
| TC-B7 | 创建 sessions 文档含 pageContent | `create_document({cname:"sessions", data:{title:"t", pageContent:"big"}})` | pageContent 被剥离，文档不含 pageContent | FP6 | P1 |
| TC-B8 | 空 data（data 为 {}） | `create_document({cname:"sessions", data:{}})` | ValueError: 创建数据不能为空 | FP6 | P1 |
| TC-B9 | update data 不含 key | `update_document({cname:"sessions", data:{title:"t"}})` | ValueError: 更新数据必须包含 key 字段 | FP7 | P0 |
| TC-B10 | update sessions messages=[] | `update_document({cname:"sessions", data:{key:<k>, messages:[]}})` | messages=[] 被丢弃，不覆盖已有 messages | FP7 | P1 |

### 2.3 异常用例

| ID | Given | When | Then | 关联 FP | 优先级 |
|----|-------|------|------|---------|--------|
| TC-E1 | cname 参数缺失 | `query_documents({})` | ValueError: 必须提供集合名称 | FP1 | P0 |
| TC-E2 | pageNum 为非数字字符串 | `query_documents({cname:"sessions", pageNum:"abc"})` | ValueError: 分页参数必须是有效的整数 | FP3 | P1 |
| TC-E3 | pageSize 为负数 | `query_documents({cname:"sessions", pageSize:-5})` | ValueError: 分页参数必须是有效的整数（`max(1, ...)` 的输入） | FP3 | P1 |
| TC-E4 | RSS 集合 link 重复创建 | 第一次 `create_document({cname:"rss", data:{link:"http://x"}})` 成功，再发同样请求 | ValueError: link 已存在 | FP6 | P0 |
| TC-E5 | 更新不存在的 key | `update_document({cname:"sessions", data:{key:"no-such-key", title:"t"}})` | ValueError: 未找到数据 | FP7 | P0 |
| TC-E6 | 删除不存在的 key | `delete_document({cname:"sessions", key:"no-such-key"})` | ValueError: 未找到数据 | FP9 | P0 |
| TC-E7 | 查询不存在的 id | `get_document_detail({cname:"sessions", id:"no-such-key"})` | ValueError: 未找到数据 | FP10 | P1 |
| TC-E8 | upsert 缺少 filter | `upsert_document({cname:"sessions", update:{a:1}})` | ValueError: Filter is required | FP8 | P1 |
| TC-E9 | upsert 缺少 update | `upsert_document({cname:"sessions", filter:{key:"x"}})` | ValueError: Update data is required | FP8 | P1 |
| TC-E10 | RSS 重复键并发创建 | 两个并发 create_document 同一 link | 一个成功一个 ValueError（依赖唯一索引兜底） | FP6 | P1 |

| ID | Given | When | Then | 关联 FP | 优先级 |

| TC-R5 | RSS 集合有 link 唯一索引 | `create_document({cname:"rss", data:{link:"http://x"}})` 并发两次 | 至少一次返回 ValueError | FP6 | P1 |

## §3 环境专项

| ID | Given | When | Then | 优先级 |
|----|-------|------|------|--------|
| TC-X1 | MongoDB 不可用 | 调用任意 data_service 方法 | 应用启动时已失败，或连接池耗尽时抛异常 | P1 |
| TC-X2 | 集合不存在（如 `nonexistent_collection`） | `query_documents({cname:"nonexistent_collection"})` | MongoDB 返回空结果（不自动创建集合） | P1 |
| TC-X3 | MongoDB 连接池耗尽 | 大量并发查询 | motor 等待队列（max 10s）或抛 `ExceededMaxWaiters` | P1 |
| TC-X4 | 超大文档（pageContent > 10MB） | `create_document({cname:"sessions", data:{...}})` | sessions 自动剥离 pageContent，不影响创建 | P1 |
