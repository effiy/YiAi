## §2 测试用例

### 2.1 正常用例

| ID | Given | When | Then | 关联 FP | 优先级 |
|----|-------|------|------|---------|--------|
| TC-N1 | 白名单含 `services.test:echo`，echo 返回 `{"msg":"ok"}` | POST `module_name="services.test", method_name="echo", parameters={}` | 返回 `{code:0, data:{"msg":"ok"}}` | FP1,FP2,FP3 | P0 |
| TC-N2 | 白名单含 `*`（全放行） | POST 任意已安装模块 | 正常执行返回 | FP1 | P1 |
| TC-N3 | 参数为合法 JSON 字符串 `'{"key":"value"}'` | POST 传入 JSON 字符串参数 | 解析为 dict，函数收到 `{"key":"value"}` | FP2 | P0 |
| TC-N4 | 目标函数为普通同步函数 | POST 调用同步函数 | 返回 JSON 结果 | FP3 | P0 |
| TC-N5 | 目标函数为 `async def` 协程 | POST 调用协程函数 | await 执行，返回 JSON 结果 | FP3 | P0 |
| TC-N6 | 沙箱开启，fs_allowlist 含目标路径 | 函数内 open() 允许路径 | 正常执行 | FP4 | P1 |
| TC-N7 | 脚本路径合法，脚本正常退出 | `run_script("/path/to/script.py")` | 返回 `{success:true, stdout:"...", returncode:0}` | FP6 | P1 |

### 2.2 边界用例

| ID | Given | When | Then | 关联 FP | 优先级 |
|----|-------|------|------|---------|--------|
| TC-B1 | module_path 为空字符串 | POST `module_name=""` | 返回 400 INVALID_PARAMS | FP1 | P0 |
| TC-B2 | function_name 为空字符串 | POST `method_name=""` | 返回 400 INVALID_PARAMS | FP1 | P0 |
| TC-B3 | parameters 为空 dict `{}` | POST `parameters={}` | 正常执行（函数以空参数调用） | FP2 | P1 |
| TC-B4 | parameters 为超大 JSON（>10KB） | POST 10KB JSON 字符串 | 解析成功，但截断记录到 500 字符 | FP2,FP7 | P1 |
| TC-B5 | 守卫深度恰好等于 max_depth=3 | 递归调用第 3 层 | 正常执行 | FP5 | P1 |
| TC-B6 | 沙箱 fs_allowlist 为空列表 | 函数尝试 open() | 所有路径被拒绝 | FP4 | P1 |

### 2.3 异常用例

| ID | Given | When | Then | 关联 FP | 优先级 |
|----|-------|------|------|---------|--------|
| TC-E1 | 白名单不含目标 `module:function` | POST 未授权模块 | 返回 403 PERMISSION_DENIED | FP1 | P0 |
| TC-E2 | parameters 为非法 JSON `"not json"` | POST 非法 JSON | 返回 400 INVALID_PARAMS "Invalid JSON" | FP2 | P0 |
| TC-E3 | parameters 为 JSON array `"[1,2,3]"` | POST JSON 数组 | 返回 400 "Parameters must be a JSON object" | FP2 | P1 |
| TC-E4 | module_path 指向不存在的模块 | POST `module_name="no.such.module"` | 返回 400 "Module or function not found" | FP3 | P0 |
| TC-E5 | function_name 在模块中不存在 | POST 存在模块但错误方法名 | 返回 400 "Module or function not found" | FP3 | P0 |
| TC-E6 | 递归调用超过 max_depth=3 | 第 4 层调用 | 返回 500 "Reentrancy depth 4 exceeds limit 3" | FP5 | P0 |
| TC-E7 | 目标函数执行时抛出异常 | 函数内 `raise ValueError("boom")` | 返回 500 "Execution failed: boom" | FP3 | P1 |
| TC-E8 | 脚本路径不存在 | `run_script("/nonexistent/script.py")` | 返回 `{success:false, ...}` | FP6 | P1 |
| TC-E9 | 脚本执行超时 | `run_script(script, timeout=0.1)` 且脚本 sleep 10s | 超时 kill，返回 "脚本执行超时" | FP6 | P1 |
| TC-E10 | 沙箱内 open() 未授权路径 | 函数 open("/etc/passwd") | 抛出 SandboxViolation | FP4 | P0 |
| TC-E11 | State Store 不可用 | MongoDB 断开时执行 | 执行成功，记录失败仅 log | FP7 | P1 |
