# CLI 工具 — 用户使用场景

## 场景 1：查看最近的状态记录

**参与者**：运维人员

**流程**：
1. 在终端执行：`python -m src.cli.state_query list --page-size 20`
2. 以 rich 表格形式展示最近 20 条状态记录
3. 显示字段：key、record_type、title、created_time 等
4. 底部显示分页信息：Total / Page

## 场景 2：按类型过滤

**参与者**：运维人员

**流程**：
1. `python -m src.cli.state_query list --record-type skill_execution`
2. 只显示 record_type 为 "skill_execution" 的记录

## 场景 3：导出为 JSON

**参与者**：数据分析人员

**流程**：
1. `python -m src.cli.state_query export --record-type skill_execution --output data.json --format json`
2. 查询所有 skill_execution 类型的记录（page_size=8000）
3. 导出为 JSON 文件
4. 可用于数据分析或备份

## 场景 4：快速查看统计

**参与者**：运维人员

**流程**：
1. `python -m src.cli.state_query stats --record-type skill_execution`
2. 输出：`Total records: 1523`

## 场景 5：获取单条记录详情

**参与者**：开发者

**流程**：
1. `python -m src.cli.state_query get <key>`
2. 以 JSON 格式打印完整记录
3. 若记录不存在，输出红色错误并 exit 1
