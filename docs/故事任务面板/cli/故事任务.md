# CLI 工具

## 故事概述

构建状态记录查询的命令行工具（CLI），基于 typer + rich 实现，支持表格/JSON/CSV 多种输出格式，提供 list、get、export、stats 四个子命令。

## 功能范围

### 状态查询 CLI (`cli/state_query.py`)

- **list**：分页查询状态记录
  - 筛选：--record-type、--tag、--title
  - 分页：--page、--page-size
  - 输出：--format table|json|csv、--output 文件
  - 表格模式使用 rich.Table 渲染

- **get**：按 key 获取单条记录并 JSON 打印

- **export**：导出查询结果到文件（json/csv），默认 page_size=8000

- **stats**：查看记录总数统计
