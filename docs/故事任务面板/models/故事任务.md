# 数据模型

## 故事概述

定义 YiAi 服务的 Pydantic 请求/响应模型和 MongoDB 集合名常量，为 API 路由层和服务层提供类型安全和数据校验。

## 功能范围

### 请求/响应模型 (`models/schemas.py`)
- **ExecuteRequest**：module_name、method_name、parameters — 动态模块执行请求
- **FileUploadRequest**：target_dir、filename、content、is_base64 — 文件上传
- **ImageUploadToOssRequest**：data_url、filename、directory — 图片上传 OSS
- **FileReadRequest / FileWriteRequest**：target_file、content、is_base64 — 文件读写
- **FileDeleteRequest / FolderDeleteRequest**：target_file、target_dir — 文件/目录删除
- **FileRenameRequest / FolderRenameRequest**：old_path/new_path、old_dir/new_dir — 重命名
- **StateRecord**：key、record_type、title、tags、data — 状态记录
- **StateQueryRequest**：record_type、tags、title_contains、created_after、created_before、分页参数
- **WeWorkWebhookRequest**：webhook_url、content — 企业微信消息
- **CleanupRequest**：dry_run、cleanup_sessions — 清理请求
- **ObserverHealth**：各组件的启用状态和运行时数据

### 集合名常量 (`models/collections.py`)
- 定义 MongoDB 集合名的字符串常量，避免硬编码
- STATE_RECORDS = "state_records"
