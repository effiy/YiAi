# 数据模型 — 测试用例评审

## 测试用例

### TC-MD-001：ExecuteRequest 必填字段校验

| 项目 | 内容 |
|------|------|
| 目标 | 缺少必填字段时 Pydantic 抛出 ValidationError |
| 步骤 | 创建 `ExecuteRequest(module_name="test")` 不传 method_name |
| 预期 | ValidationError，提示 method_name 必填 |

### TC-MD-002：FileUploadRequest 默认值

| 项目 | 内容 |
|------|------|
| 目标 | is_base64 默认 False |
| 步骤 | 创建 `FileUploadRequest(filename="test.txt", content="hello")` |
| 预期 | request.is_base64 == False |

### TC-MD-003：StateRecord exclude_unset

| 项目 | 内容 |
|------|------|
| 目标 | model_dump(exclude_unset=True) 不包含未设置的字段 |
| 步骤 | 创建 `StateRecord(record_type="test")` 调用 model_dump(exclude_unset=True) |
| 预期 | 结果只包含 record_type，不包含 title/tags/data 等 |

### TC-MD-004：ImageUploadToOssRequest 字段校验

| 项目 | 内容 |
|------|------|
| 目标 | data_url 为空时校验 |
| 步骤 | 创建 `ImageUploadToOssRequest(data_url="", filename="test.png")` |
| 预期 | data_url 为空字符串，应在业务层被拒绝 |

### TC-MD-005：分页参数边界

| 项目 | 内容 |
|------|------|
| 目标 | page_num 最小为 1 |
| 步骤 | 查询接口传入 page_num=0 |
| 预期 | FastAPI 校验失败（ge=1） |

### TC-MD-006：集合名配置一致性

| 项目 | 内容 |
|------|------|
| 目标 | Settings 中集合名与 config.yaml 一致 |
| 步骤 | 对比 config.yaml 中 collection_* 与 Settings 类对应 Field 默认值 |
| 预期 | 完全一致 |
