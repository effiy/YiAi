# 项目总览 — 测试用例评审

### TC-OV-001：项目画像完整性

| 项目 | 内容 |
|------|------|
| 目标 | 验证项目画像 8 项信息完整且与实际一致 |
| 步骤 | 1. 检查项目名 2. 检查项目类型 3. 检查版本号 4. 检查仓库 URL 5. 检查入口文件存在 6. 检查架构描述 7. 检查数据库类型 8. 检查 AI 引擎配置 |
| 预期 | 8 项全部与实际代码和配置一致 |
