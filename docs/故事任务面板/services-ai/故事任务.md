# AI 对话服务

## 故事概述

基于 Ollama 的 AI 对话服务，提供同步/流式对话生成、多模态图片识别（qwen3-vl）和模型列表查询能力。通过模块执行引擎暴露为可远程调用的服务函数。

## 功能范围

### Ollama 客户端封装 (`services/ai/chat_service.py`)

- **OllamaService 类**：封装 ollama Python 客户端
  - 支持 basic auth（username:password 格式）
  - `generate_response()`：同步对话生成，支持最大重试 2 次
  - `list_models()`：获取服务端可用模型列表

- **chat() 异步函数**：结构化对话接口
  - 参数：system（系统提示词）、user（用户输入）、model（模型名，默认 qwen3.5）、stream（是否流式）、images（图片列表）
  - 非流式：线程池执行 generate_response，返回完整结果
  - 流式：创建异步生成器，通过队列跨线程传递 token
  - 多模态：有 images 参数时自动切换为 qwen3-vl 模型

- **list_ollama_models()**：获取可用模型列表（线程池执行）

- **图片处理**：
  - 支持 data: URL（base64）和 HTTP URL 两种图片格式
  - HTTP URL 图片并发下载（信号量限制 4 并发）
  - 单张最大 10MB，256KB 分块读取

## 验收标准
- [ ] 同步对话正常返回 AI 响应
- [ ] 流式对话通过 SSE 逐 token 推送
- [ ] 图片输入正确传递给多模态模型
- [ ] HTTP URL 图片正确下载并解码
- [ ] 失败自动重试最多 2 次
- [ ] 模型列表查询正常返回
