# AI 对话服务 — 测试用例评审

## 测试用例

### TC-AI-001：文本对话同步调用

| 项目 | 内容 |
|------|------|
| 目标 | 同步模式正常返回 AI 响应 |
| 前置 | Ollama 服务可用，qwen3.5 模型已安装 |
| 步骤 | 调用 chat({"user": "Hello"}) |
| 预期 | 返回 {"success": true, "model": "qwen3.5", "message": "..."} |

### TC-AI-002：自定义模型

| 项目 | 内容 |
|------|------|
| 目标 | 使用指定模型对话 |
| 步骤 | 调用 chat({"user": "Hello", "model": "llama3"}) |
| 预期 | 返回结果 model 为 "llama3" |

### TC-AI-003：流式对话

| 项目 | 内容 |
|------|------|
| 目标 | 流式模式返回异步生成器 |
| 步骤 | 调用 chat({"user": "Hello", "stream": true}) |
| 预期 | 返回 async generator，逐 token yield |

### TC-AI-004：多模态图片输入

| 项目 | 内容 |
|------|------|
| 目标 | 有图片时自动切换 qwen3-vl |
| 前置 | qwen3-vl 模型已安装 |
| 步骤 | 调用 chat({"user": "描述这张图片", "images": ["data:image/png;base64,..."]}) |
| 预期 | model 为 "qwen3-vl"，正常返回图片描述 |

### TC-AI-005：HTTP URL 图片下载

| 项目 | 内容 |
|------|------|
| 目标 | HTTP URL 图片正确下载并传给模型 |
| 步骤 | 调用 chat({"user": "分析图片", "images": ["https://example.com/photo.jpg"]}) |
| 预期 | 图片被下载、解码，正常传入模型 |

### TC-AI-006：重试机制

| 项目 | 内容 |
|------|------|
| 目标 | 首次失败自动重试 |
| 前置 | 模拟首次调用失败 |
| 步骤 | 1. Mock ollama.chat 首次抛异常 2. 第二次正常 |
| 预期 | 最终返回成功结果（max_retries=2） |

### TC-AI-009：大图片拒绝

| 项目 | 内容 |
|------|------|
