# 数据库服务层 — 测试用例评审

## 测试用例

### TC-DB-001：基本查询

| 项目 | 内容 |
|------|------|
| 目标 | 无过滤条件的基本分页查询 |
| 步骤 | query_documents({"collection_name": "rss", "pageNum": 1, "pageSize": 10}) |
| 预期 | 返回 list 长度 ≤ 10，total 正确 |

### TC-DB-002：Key 精确查询

| 项目 | 内容 |
|------|------|
| 目标 | key 字段精确匹配 |
| 步骤 | query_documents({"cname": "sessions", "key": "abc-123"}) |
| 预期 | 返回 key == "abc-123" 的文档 |

### TC-DB-003：字符串模糊搜索

| 项目 | 内容 |
|------|------|
| 目标 | 单个搜索词模糊匹配 |
| 步骤 | query_documents({"cname": "sessions", "filter": {"title": "test"}}) |
| 预期 | 返回 title 含 "test"（大小写不敏感）的文档 |

### TC-DB-004：多词搜索

| 项目 | 内容 |
|------|------|
| 目标 | 逗号分隔多词 OR 搜索 |
| 步骤 | query_documents({"cname": "rss", "filter": {"title": "AI,ML"}}) |
| 预期 | 构建 $or: [{title: /AI/}, {title: /ML/}]，返回匹配任意词的文档 |

### TC-DB-005：日期范围查询

| 项目 | 内容 |
|------|------|
| 目标 | isoDate 范围查询 |
| 步骤 | query_documents({"cname": "rss", "filter": {"isoDate": "2024-01-01,2024-01-31"}}) |
| 预期 | 构建多格式日期正则，返回范围内文档 |

### TC-DB-006：数值范围查询

| 项目 | 内容 |
|------|------|
| 目标 | 数值 $gte/$lt 范围 |
| 步骤 | query_documents({"cname": "test", "filter": {"score": [60, 90]}}) |
| 预期 | 构建 {"score": {"$gte": 60, "$lt": 90}} |

### TC-DB-007：创建文档

| 项目 | 内容 |
|------|------|
| 目标 | 创建文档自动生成 key 和时间戳 |
| 步骤 | create_document({"cname": "test", "data": {"title": "hello"}}) |
| 预期 | 返回 key，文档包含 key、createdTime、updatedTime、order |

### TC-DB-008：更新文档

| 项目 | 内容 |
|------|------|
| 目标 | 部分字段更新 |
| 步骤 | 1. 创建文档 2. update_document({"cname": "test", "data": {"key": "...", "title": "updated"}}) |
| 预期 | title 更新，createdTime 不变，updatedTime 更新 |

### TC-DB-009：删除文档

| 项目 | 内容 |
|------|------|
| 目标 | 按 key 删除 |
| 步骤 | 1. 创建文档 2. delete_document({"cname": "test", "key": "..."}) 3. 查询该 key |
| 预期 | 删除返回 deleted:True，再次查询返回 404 |

### TC-DB-010：rss link 唯一性

| 项目 | 内容 |
|------|------|
| 目标 | 重复 link 创建被拒绝 |
| 步骤 | 1. 创建 rss 文档 link="http://a.com" 2. 再次创建相同 link |
| 预期 | 第二次创建抛出 ValueError |

### TC-DB-011：sessions pageContent 排除

| 项目 | 内容 |
|------|------|
| 目标 | 查询 sessions 时不返回 pageContent |
| 步骤 | query_documents({"cname": "sessions"}) |
| 预期 | 返回的文档不包含 pageContent 字段 |
