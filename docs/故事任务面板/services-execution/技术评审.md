# 模块执行引擎 — 后端技术评审

## 架构设计

### 执行引擎核心

```
execute_module(module_path, function_name, parameters)
    │
    ├── _acquire_guard()
    │   └── contextvar _reentrancy_depth.get() >= max_depth → raise
    │
    ├── _check_whitelist()
    │   └── "*" in EXEC_ALLOWLIST or f"{module}:{func}" in EXEC_ALLOWLIST
    │
    ├── parse_parameters()
    │   ├── dict → 直接返回
    │   └── str → json.loads → 校验必须为 dict
    │
    ├── _import_target_function()
    │   ├── importlib.import_module(module_path)
    │   └── getattr(module, function_name)
    │
    ├── 函数类型检测
    │   ├── asyncgenfunction → 返回生成器（不自执行）
    │   ├── generatorfunction → 返回生成器（不自执行）
    │   └── coroutinefunction / 普通函数 → _run_function()
    │
    └── _run_function()
        ├── sandbox_enabled → sandbox_context(fs, network)
        └── await / call target_function(parameters_dict)
```

### 脚本执行

```
run_script(script_path, timeout=300)
    ├── asyncio.create_subprocess_exec('python3', script_path)
    ├── asyncio.wait_for(process.communicate(), timeout)
    ├── 超时 → process.kill()
    └── 返回 {success, stdout, stderr, returncode}
```

### 执行记录（SkillRecorder）

```
_record_execution()
    ├── _get_recorder() → SkillRecorder (lazy import)
    ├── recorder.record_async(
    │       skill_name="module:function",
    │       status="success"/"failed",
    │       duration_ms=...,
    │       input_summary=... (截断 500 字符),
    │       output_summary=... (截断 500 字符),
    │       error_message=...
    │   )
    └── best-effort: 记录失败不影响主流程
```

## 关键决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 白名单 | 模块:函数 格式 + * 通配 | 精确控制可执行范围 |
| 参数格式 | JSON 字符串或 dict | 兼容 GET URL 参数和 POST JSON |
| 生成器处理 | 直接返回，路由层处理 | 避免引擎层消费生成器导致无法流式传输 |
| 脚本超时 | asyncio.wait_for + kill | 保证进程资源释放 |
| 执行记录 | best-effort 异步记录 | 不影响主流程性能 |
| 沙箱集成 | 可选 + 上下文管理器 | 仅在配置启用时生效 |

## 数据流

```
API Request (module_name, method_name, parameters)
    ↓
execute_module()
    ├── [Guard] 深度检查
    ├── [Whitelist] 权限检查
    ├── [Parser] JSON → dict
    ├── [Import] importlib → getattr
    ├── [Sandbox] (可选) sandbox_context
    ├── [Execute] await/call target_function(params)
    ├── [Record] SkillRecorder.record_async()
    └── [Guard] 释放深度
    ↓
Result / Generator / AsyncGenerator → API Response
```
