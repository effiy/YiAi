# 模块执行引擎

## 故事概述

构建受控的模块执行引擎，提供白名单校验、参数解析、同步/异步函数调用、脚本执行、Observer 沙箱集成和重入守卫功能。是 `/` API 路由的核心执行器。

## 功能范围

### 执行引擎 (`services/execution/executor.py`)

- **白名单校验**：`_check_whitelist()`，格式 `module_path:function_name`，支持 `*` 通配全允许
- **参数解析**：`parse_parameters()`，支持 dict 和 JSON 字符串
- **动态导入**：`_import_target_function()`，`importlib.import_module` + `getattr`
- **函数执行**：`_run_function()`，自动检测同步/异步，可选沙箱上下文包装
- **脚本执行**：`run_script()`，`asyncio.create_subprocess_exec` + 超时控制（默认 300s）
- **生成器/异步生成器支持**：检测 asyncgen 和 generator，直接返回供路由层流式处理
- **执行记录**：`_record_execution()`，通过 SkillRecorder 记录执行状态、耗时、错误（best-effort）
- **重入守卫集成**：`_acquire_guard()` / `_release_guard()`，contextvar 追踪深度

### 执行流程

```
execute_module(module_path, function_name, parameters)
    ├── _acquire_guard()         # 深度检查
    ├── _check_whitelist()       # 白名单校验
    ├── parse_parameters()       # 参数解析
    ├── _import_target_function() # 动态导入
    ├── _run_function()          # 沙箱 + 同步/异步执行
    └── _record_execution()      # 状态记录（finally）
    └── _release_guard()         # 释放深度标记（finally）
```

## 验收标准
- [ ] 白名单校验正确拦截非授权模块/函数
- [ ] JSON 参数正确解析为 dict
- [ ] 同步和异步函数均能正确执行
- [ ] 脚本执行超时正确杀死进程
- [ ] 沙箱上下文正确包装目标函数
- [ ] 重入深度超限抛出异常
- [ ] 执行记录在成功和失败时均能写入
