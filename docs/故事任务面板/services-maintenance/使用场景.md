# 维护服务 — 用户使用场景

## 场景 1：全量获取 sessions

**参与者**：维护 API 路由

**流程**：
1. 维护接口 `cleanup_unused_images` 调用 `get_all_sessions()`
2. 返回所有 sessions 文档
3. 遍历文档提取图片引用
4. 与 static 目录实际图片对比
5. 找出未引用的图片

## 场景 2：清理无效 session

**参与者**：维护 API 路由

**流程**：
1. 维护接口检测到某 session 引用了不存在的图片
2. 调用 `delete_session_by_key(session_key)`
3. 删除该 session
4. 返回清理计数

## 场景 3：空集合查询

**参与者**：维护 API 路由

**流程**：
1. sessions 集合为空时调用 `get_all_sessions()`
2. 返回空列表
3. 维护流程正常结束（无图片引用 → 所有图片均为未引用）
