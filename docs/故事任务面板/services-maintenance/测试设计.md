# 维护服务 — 测试用例评审

## 测试用例

### TC-MN-001：获取所有 sessions

| 项目 | 内容 |
|------|------|
| 目标 | 全量返回 sessions 文档 |
| 前置 | sessions 集合有 3 条记录 |
| 步骤 | get_all_sessions() |
| 预期 | 返回列表长度 3 |

### TC-MN-002：空集合查询

| 项目 | 内容 |
|------|------|
| 目标 | 空集合返回空列表 |
| 前置 | sessions 集合为空 |
| 步骤 | get_all_sessions() |
| 预期 | 返回 [] |

### TC-MN-003：删除存在的 session

| 项目 | 内容 |
|------|------|
| 目标 | 按 key 删除存在记录 |
| 前置 | session key="test-123" 存在 |
| 步骤 | delete_session_by_key("test-123") → 再次查询 |
| 预期 | 返回 1，记录被删除 |

### TC-MN-004：删除不存在的 session

| 项目 | 内容 |
|------|------|
| 目标 | 不存在的 key 返回 0 |
| 步骤 | delete_session_by_key("nonexistent") |
| 预期 | 返回 0 |
