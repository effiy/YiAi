# 静态文件管理

## 故事概述

构建静态文件上传与管理服务，支持 ZIP 文件安全上传/解压到 static 目录，以及通过模块执行引擎调用的本地 ZIP 解压功能。

## 功能范围

### ZIP 上传与解压 (`services/static/static_files.py`)

- **upload_and_unzip()**：上传 ZIP 并安全解压
  - 格式校验（仅 .zip）
  - 大小限制（config.yaml 配置，默认 100MB）
  - 路径安全校验（`_is_safe_path`：拒绝 `..`、绝对路径）
  - ZIP 文件名编码兼容（UTF-8 / cp437 回退）
  - 自动检测并剥离公共根目录（`_find_common_root`）
  - 安全解压：逐文件验证路径安全性
  - 临时文件自动清理（finally 块）

### 本地 ZIP 解压 (`services/static/archive_service.py`)

- **unzip_from_path()**：解压本地 ZIP 到 static 目录
  - 通过执行引擎调用
  - 参数：file_path（本地绝对路径）、project_id（可选）
  - 自动创建目标目录

## 验收标准
- [ ] ZIP 上传正确解压到目标目录
- [ ] 非 ZIP 文件被拒绝
- [ ] 超大文件被拒绝
- [ ] 路径遍历攻击被阻止
- [ ] 公共根目录自动剥离
- [ ] 临时文件正确清理
