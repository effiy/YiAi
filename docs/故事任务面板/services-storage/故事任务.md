# OSS 存储服务

## 故事概述

构建阿里云 OSS（对象存储服务）集成层，提供文件上传/删除、标签管理、文件信息维护与列表查询能力，支持配置驱动和优雅降级。

## 功能范围

### OSS 客户端 (`services/storage/oss_client.py`)

- **OSSConfig**：从 Settings 加载 OSS 连接参数（access_key、secret_key、endpoint、bucket）
- **get_bucket()**：构建 oss2.Bucket 客户端
- **build_oss_url()**：生成可访问的 HTTPS URL

- **文件上传**：
  - `upload_file_to_oss(file, directory)`：UploadFile → OSS
  - `upload_bytes_to_oss(content, filename, directory)`：bytes → OSS
  - 文件扩展名白名单（可配置，默认图片+文档类型）
  - 文件大小限制（可配置，默认 50MB）
  - 自动生成时间戳文件名

- **文件删除**：
  - `delete_oss_file(object_name)`：删除 OSS 文件 + 清理关联标签和信息

- **标签管理**：
  - `set_file_tags(object_name, tags)`：设置/更新标签（去重、upsert）
  - `get_file_tags(object_name)`：获取标签列表
  - `delete_file_tags(object_name)`：删除所有标签
  - `get_all_tags()`：聚合所有标签及使用计数

- **文件信息**：
  - `update_file_info(object_name, title, description)`：更新标题/描述
  - `get_file_info(object_name)`：获取标题/描述

- **文件列表**：
  - `list_files(directory, tags)`：列出目录下文件（支持标签过滤），返回基础元数据

## 验收标准
- [ ] 文件正确上传到 OSS 并返回可访问 URL
- [ ] 不支持的文件类型被拒绝
- [ ] 文件删除同时清理关联数据
- [ ] 标签 CRUD 正常
