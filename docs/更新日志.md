## [0.0.1] - 2026-03-11

### 新增
- 初始项目提交
- 基于 FastAPI 的 AI 服务后端基础架构
- 核心功能模块：
  - RSS 源管理
  - 文件上传与存储（OSS + 本地存储）
  - AI 聊天服务（Ollama 集成）
  - 动态模块执行引擎
  - 企业微信集成
- MongoDB 数据库集成
- 配置系统（YAML + 环境变量）
- 统一响应格式和异常处理
- 可选的令牌认证中间件
