# 核心功能

本目录包含 YiAi 项目所有核心功能的详细说明。

---

## 📋 功能列表

- [AI 聊天服务](./AI聊天服务.md) - 集成 Ollama 本地 AI 模型，支持多轮对话和历史记录
- [RSS 源管理](./RSS源管理.md) - 自动定时抓取 RSS 源内容，支持多源管理
- [文件上传与存储](./文件上传与存储.md) - 支持 OSS 和本地存储双重方案，完整的文件管理功能
- [动态模块执行引擎](./动态模块执行引擎.md) - 通过 REST API 动态执行 Python 模块方法
- [安全与认证](./安全与认证.md) - 可选的令牌认证，统一的异常处理
