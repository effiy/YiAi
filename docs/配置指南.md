## 配置方式

YiAi 支持两种配置方式：**YAML 配置文件** 和 **环境变量**。

### YAML 配置文件

主要配置文件为项目根目录下的 `config.yaml`。

### 环境变量覆盖

所有配置项都可以通过环境变量覆盖。环境变量命名规则为：
- 大写字母
- 下划线分隔
- 嵌套的 YAML 键会被扁平化

例如：
- YAML 路径 `server.host` → 环境变量 `SERVER_HOST`
- YAML 路径 `mongodb.url` → 环境变量 `MONGODB_URL`
- YAML 路径 `oss.access_key` → 环境变量 `OSS_ACCESS_KEY`

## 服务器配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `server.host` | string | `"0.0.0.0"` | 服务器监听地址 |
| `server.port` | int | `8000` | 服务器监听端口 |
| `server.reload` | bool | `true` | 是否启用自动重载（开发模式） |
| `uvicorn.limit_concurrency` | int | `1000` | 最大并发连接数 |
| `uvicorn.limit_max_requests` | int | `10000` | 最大请求数后重启 |
| `uvicorn.timeout_keep_alive` | int | `5` | Keep-Alive 超时时间（秒） |

**示例：**
```yaml
server:
  host: "127.0.0.1"
  port: 8000
  reload: true
```

## CORS 配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `cors.origins` | list | `["*"]` | 允许的跨域源列表 |
| `cors.allow_any_origin` | bool | `true` | 是否允许任意源 |

**示例：**
```yaml
cors:
  origins:
    - "https://example.com"
    - "https://app.example.com"
  allow_any_origin: false
```

## 分页配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|

| `pagination.max_size` | int | `8000` | 最大分页大小 |

## 静态文件配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `static.base_dir` | string | `"./static"` | 静态文件存储目录 |
| `static.base_url` | string | `"https://api.effiy.cn/static"` | 静态文件访问 URL 前缀 |

**示例：**
```yaml
static:
  base_dir: "/var/www/static"
  base_url: "https://your-domain.com/static"
  max_zip_size_mb: 100
```

## OSS 存储配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `oss.access_key` | string | `""` | OSS 访问密钥 ID |
| `oss.secret_key` | string | `""` | OSS 访问密钥 Secret |
| `oss.endpoint` | string | `""` | OSS 接入点（Endpoint） |
| `oss.bucket` | string | `""` | OSS 存储空间名称 |
| `oss.domain` | string | `""` | OSS 自定义域名 |
| `oss.max_file_size_mb` | int | `50` | 最大文件大小（MB） |
| `oss.allowed_extensions` | list | 见下方 | 允许的文件扩展名列表 |

**允许的文件扩展名默认值：**
```yaml
- ".jpg"
- ".jpeg"
- ".png"
- ".gif"
- ".pdf"
- ".doc"
- ".docx"
- ".epub"
- ".md"
```

**示例：**
```yaml
oss:
  access_key: "your-access-key-id"
  secret_key: "your-access-key-secret"
  endpoint: "oss-cn-hangzhou.aliyuncs.com"
  bucket: "your-bucket-name"
  domain: "https://files.your-domain.com"
  max_file_size_mb: 100
  allowed_extensions:
    - ".jpg"
    - ".png"
    - ".pdf"
```

## RSS 配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `rss.scheduler_enabled` | bool | `true` | 是否启用 RSS 调度器 |
| `rss.scheduler_interval` | int | `3600` | 抓取间隔（秒） |

**示例：**
```yaml
rss:
  scheduler_enabled: true
  scheduler_interval: 1800  # 30 分钟
```

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `startup.init_database` | bool | `true` | 应用启动时是否初始化数据库连接 |

## 中间件配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `middleware.auth_enabled` | bool | `false` | 是否启用认证中间件 |
| `middleware.auth_token` | string | `""` | 认证令牌 |

**示例：**
```yaml
middleware:
  auth_enabled: true
  auth_token: "your-secret-token-here"
```

**环境变量覆盖：**
认证令牌也可以通过 `API_X_TOKEN` 环境变量设置，优先级高于配置文件。

## 模块执行配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `module.allowlist` | list | `["*"]` | 允许执行的模块白名单 |

**示例：**
```yaml
module:
  allowlist:
    - "services.storage.oss_client"
    - "services.rss.scheduler"
    - "services.ai.ollama_client"
```

使用 `["*"]` 表示允许执行所有模块。

## 日志配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `logging.level` | string | `"INFO"` | 日志级别（DEBUG/INFO/WARNING/ERROR） |
| `logging.format` | string | 见下方 | 日志格式 |
| `logging.datefmt` | string | `"%Y-%m-%d %H:%M:%S"` | 日期时间格式 |
| `logging.propagate_uvicorn` | bool | `false` | 是否传播 Uvicorn 日志 |

**默认日志格式：**
```
"%(asctime)s - %(name)s - %(levelname)s - %(message)s"
```

**示例：**
```yaml
logging:
  level: "DEBUG"
  format: "%(asctime)s - %(name)s - %(levelname)s - %(message)s"
  datefmt: "%Y-%m-%d %H:%M:%S"
  propagate_uvicorn: false
```
